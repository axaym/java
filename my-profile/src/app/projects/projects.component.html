PROJECT DESCRIPTION:

My State Street is a web based near real time dashboard application provides status updates to clients for Trading, Reconciliation and various reporting. There are multiple high-level modules with multiple sub modules as listed: Trade management Dashboard, Investor Services dashboard, Received Trade Dashboard, Regulatory Resource Center, Report Access Inbox, Report Center, Interactive Views.

CLIENT DESCRIPTION:

State Street Corporation, known as State Street, is a US-based international financial services holding company. State Street was founded in 1792 and is the second oldest financial institution in the United States.
